# Licitações — Guia de Uso

## O que é?

O módulo de Licitações gerencia **processos licitatórios** que sua empresa participa como fornecedora. Permite acompanhar todas as etapas: desde a publicação do edital até o resultado (vencedor ou não).

---

## Fluxo Completo

```
Edital → Cadastro → Proposta → Julgamento → Resultado → Contrato
```

---

## Passo a Passo

### 1. Cadastrar uma Licitação

1. Acesse **Vendas > Licitações**
2. Clique em **+ Nova Licitação**
3. Preencha os dados:
   - **Número/Ano**: identificação do processo (ex: "001/2026")
   - **Órgão**: nome do órgão público licitante
   - **Modalidade**: Pregão, Concorrência, Tomada de Preços, Convite, Concurso
   - **Objeto**: descrição resumida do que está sendo licitado
   - **Data de Abertura**: data da sessão pública
   - **Valor Estimado**: valor da proposta
   - **Status**: Rascunho, Publicada, Em Andamento, Adjudicada, Homologada, Perdida, Cancelada

### 2. Adicionar Itens do Edital

1. Na licitação, clique em **Adicionar Item**
2. Selecione o **produto** (campo lookup)
3. Informe: quantidade, valor unitário, descrição técnica
4. Repita para cada item do edital

### 3. Acompanhar Status

| Status | Descrição |
|--------|-----------|
| **Rascunho** | Em preparação |
| **Publicada** | Edital publicado, aguardando sessão |
| **Em Andamento** | Sessão em andamento |
| **Adjudicada** | Sua empresa venceu (aguardando homologação) |
| **Homologada** | Processo finalizado com vitória |
| **Perdida** | Outra empresa venceu |
| **Cancelada** | Licitação cancelada pelo órgão |

### 4. Registrar Resultado

1. Após a sessão, atualize o status
2. Se **vencedora**: sistema pode gerar contrato automaticamente
3. Se **perdedora**: registre o motivo e valor do vencedor

---

## Dicas e Boas Práticas

- **Acompanhe prazos**: edital tem datas rígidas
- **Documente tudo**: anexe o edital em PDF
- **Registre concorrentes**: saber quem ganhou ajuda em futuras estratégias
- **Margem de lucro**: considere as despesas de participação (taxas, garantias)

---

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Órgão não aparece no lookup | Cadastre o órgão como cliente primeiro |
| Edital anexado grande demais | Comprima em ZIP antes de anexar |
| Proposta perdeu por detalhe técnico | Revise os requisitos técnicos do edital |
