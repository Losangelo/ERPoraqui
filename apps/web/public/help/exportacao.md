# Exportação de Dados — Guia de Uso

## O que é?

O recurso de **Exportação** permite baixar dados do sistema em vários formatos: **CSV, XLSX, JSON e PDF**. Disponível nos principais relatórios e páginas de listagem.

---

## Formatos Disponíveis

| Formato | Ideal para | Abre em |
|---------|------------|---------|
| **CSV** | Tratar dados no Excel ou Google Sheets | Excel, Google Sheets, LibreOffice |
| **XLSX** | Planilha formatada para imprimir ou enviar | Excel, Google Sheets |
| **JSON** | Integrações técnicas, APIs, desenvolvedores | Qualquer editor de texto |
| **PDF** | Relatórios prontos para imprimir ou assinar | Adobe Reader, navegador |

---

## Onde Encontrar

O botão de exportação está presente em:
- **DRE** (Demonstração de Resultados)
- **Fluxo de Caixa**
- **Contas a Receber**
- **Contas a Pagar**
- **Relatórios Fiscais**
- **Relatórios Personalizados** (Report Engine)

## Dicas e Truques

### CSV no Excel
- No Excel português, o separador padrão é **ponto e vírgula (;)**
- No Excel inglês, o separador é **vírgula (,)**
- Se o CSV abrir tudo na mesma coluna, use **Dados > De Texto/CSV** e escolha o separador

### XLSX
- Planilha já formatada com cabeçalho, cores e valores em R$
- Pronta para imprimir — só abrir e mandar
- Ideal para enviar para clientes ou gerência

### PDF
- Relatório visual pronto — use para assinar, carimbar, arquivar
- Ideal para prestação de contas
- Pode ser enviado por email diretamente

### JSON
- Formato técnico para desenvolvedores
- Use se precisar integrar com outro sistema
- Mantém a estrutura original dos dados

---

## Boas Práticas

- **CSV** para analisar dados (gráficos, tabela dinâmica, filtros)
- **XLSX** para compartilhar relatórios prontos
- **PDF** para documentos formais
- **JSON** para integrações técnicas
- Sempre **filtre antes de exportar** — dados demais = arquivo enorme
- Relatórios mensais: exporte, salve em pastas organizadas por mês
