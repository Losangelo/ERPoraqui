# Inventário de Estoque — Guia de Uso

## O que é?

O **Inventário** é a contagem física dos produtos para verificar se o estoque do sistema corresponde ao estoque real. Divergências são ajustadas após a contagem.

---

## Quando Fazer Inventário

- **Mensal**: recomendado para controle básico
- **Trimestral/Semestral**: para empresas com grande volume
- **Anual**: obrigatório para algumas empresas (fechamento contábil)
- **Rotativo**: conte um grupo de produtos por vez, continuamente
- **Extraordinário**: quando suspeitar de divergência

---

## Passo a Passo

### 1. Criar Inventário

1. Acesse **Estoque > Inventário**
2. Clique em **+ Novo Inventário**
3. Defina:
   - **Tipo**: Geral (todos produtos) ou Parcial (selecionados)
   - **Filial**: qual estoque será contado
   - **Data**: data da contagem
4. Salve

O sistema carrega todos os produtos com as quantidades atuais do sistema.

### 2. Realizar a Contagem

1. Na lista, para cada produto:
   - Conte fisicamente no estoque
   - Digite a **quantidade contada**
   - Marque como "Contado"
2. Você pode pular produtos que não queira contar agora

### 3. Verificar Divergências

1. Clique em **Verificar Divergências**
2. O sistema mostra:
   - Produtos com diferença
   - Quantidade no sistema vs. contada
   - Valor da divergência

### 4. Conciliar (Ajustar)

1. Selecione os itens com divergência
2. Clique em **Conciliar**
3. Escolha a ação:
   - **Ajustar Estoque**: sistema atualiza para a quantidade contada
   - **Manter Estoque**: ignora a divergência (pode refazer depois)
4. Confirme

---

## Dicas e Boas Práticas

- **Conte em dupla** — uma pessoa conta, outra registra
- **Produtos parados**: aproveite para verificar validade e condição
- **Investigue divergências grandes** antes de ajustar
- **Exporte a lista** para contagem offline em papel
- **Faça inventário rotativo** para não parar o estoque

---

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Produto não aparece no inventário | Verifique se o tipo (Geral/Parcial) inclui o produto |
| Diferença muito grande | Pode ser erro de contagem — reconte |
| Já ajustei mas estoque continua errado | Verifique movimentações após o ajuste |
