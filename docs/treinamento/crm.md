# CRM (Customer Relationship Management) — Guia de Uso

## O que é?

O CRM é o módulo de **gestão de relacionamento com o cliente**, focado em vendas e marketing. Ele organiza todo o funil de vendas: desde o primeiro contato até o fechamento do negócio e o pós-venda.

---

## Conceitos Essenciais

| Termo | Significado |
|-------|-------------|
| **Pipeline** | Funil de vendas — representa as etapas do processo comercial |
| **Oportunidade** | Um negócio em potencial (lead qualificado) |
| **Etapa** | Fase do funil (ex: Prospecção, Proposta, Negociação, Fechamento) |
| **Tarefa** | Ação a ser feita (ligar, enviar email, visitar) |
| **Interação** | Registro de contato com o cliente (ligação, reunião, WhatsApp) |
| **Visão 360º** | Perfil completo do cliente com tudo que já aconteceu |
| **Quote-to-Cash** | Conversão automática de oportunidade em pedido |

---

## Pipeline de Vendas

```
Prospecção → Qualificação → Proposta → Negociação → Fechado (Ganho)
                                                       ↓
                                                  Fechado (Perdido)
```

| Etapa | O que fazer |
|-------|-------------|
| **Prospecção** | Identificar potenciais clientes — primeiro contato |
| **Qualificação** | Entender a necessidade, orçamento, prazo |
| **Proposta** | Enviar orçamento ou proposta comercial |
| **Negociação** | Ajustar condições, prazo, descontos |
| **Fechado (Ganho)** | Venda concluída — vira pedido |
| **Fechado (Perdido)** | Cliente não fechou — registrar motivo |

---

## Funcionalidades Principais

### 1. Oportunidades
- Crie oportunidades para cada negócio em andamento
- Atribua ao **vendedor responsável**
- Defina o **valor estimado** e **probabilidade de fechamento**
- Arraste e solte entre as etapas do pipeline (Kanban)

### 2. Tarefas
- Vincule tarefas a oportunidades ou clientes
- Tipos: **Ligação**, **Email**, **Reunião**, **Visita**, **Lembrete**
- Defina **data e hora** — o sistema alerta no dashboard
- Marque como concluída ao finalizar

### 3. Interações
- Registre **cada contato** com o cliente
- Anote o **resumo da conversa** e **próximos passos**
- Histórico completo: quem falou o quê e quando

### 4. Visão 360º do Cliente
- Na tela do cliente, veja **tudo junto**:
  - Pedidos já feitos
  - Contratos ativos
  - Garantias e devoluções
  - Interações registradas
  - Oportunidades abertas
  - Contas a receber
- Ideal para atendimento: um só lugar com todo o histórico

### 5. Quote-to-Cash
- Uma oportunidade na etapa "Fechado (Ganho)" pode ser **convertida em pedido**
- O sistema copia cliente, produtos, valores da proposta
- Gera o pedido de venda automaticamente
- Do orçamento ao dinheiro no banco em poucos cliques

### 6. Campanhas (Premium)
- Crie campanhas de marketing (email, SMS, WhatsApp)
- Segmente clientes por perfil, compras anteriores, região
- Acompanhe taxas de abertura e resposta

## Dicas e Truques

### Funil Saudável
- **Mantenha oportunidades em todas as etapas** — não só no início ou no fim
- **Probabilidade**: seja realista — 10% é prospecção, 90% é negociação avançada
- **Valor estimado**: melhor subestimar que superestimar

### Tarefas Diárias
- Crie tarefas para **cada ação que precisa tomar**
- Ex: "Ligar para João amanhã às 10h para apresentar proposta"
- Revise as tarefas **todo início de dia**

### Visão 360º
- **Sempre consulte** o histórico do cliente antes de ligar
- Cliente reclamou no mês passado? Saiba antes de oferecer outro produto
- Cliente devolveu um produto? Verifique o motivo antes de vender novamente

### Quote-to-Cash
- Use para agilizar o processo: **oportunidade → pedido → NF-e → recebimento**
- Evita redigitar dados — tudo já está na oportunidade
- Ideal para vendas B2B com ciclo mais longo

### Motivo de Perda
- Sempre registre **por que perdeu** a venda:
  - Preço? Concorrente? Prazo? Relacionamento?
- Use esses dados para melhorar as próximas propostas

---

## Boas Práticas

- **Alimente o CRM diariamente** — ferramenta parada não gera resultado
- **Registre toda interação** — mesmo as rápidas
- **Mantenha o pipeline atualizado** — arraste oportunidades entre etapas
- **Use tarefas** como sua lista de afazeres do dia
- **Revise as oportunidades perdidas** semanalmente para aprender
- **Quote-to-Cash**: integre CRM com vendas para não perder dados

## Exemplo Prático

**Cenário:** Vendedor Maria prospecta a "Empresa ABC" para vender software.

1. **Oportunidade criada**: Empresa ABC, R\$ 15.000, etapa Prospecção
2. **Tarefa**: "Ligar para João (contato) amanhã 09h"
3. **Ligação realizada**: Maria liga, João demonstrou interesse
4. **Interação registrada**: "João pediu proposta por email"
5. **Arrasta** oportunidade para etapa **Proposta**
6. **Envia proposta**: orçamento de R\$ 15.000
7. **Negociação**: cliente pede 10% de desconto
8. **Ajusta valor** para R\$ 13.500
9. **Arrasta** para etapa **Negociação**
10. Cliente aceita → arrasta para **Fechado (Ganho)**
11. **Quote-to-Cash**: converte automaticamente em pedido de venda
12. Maria comemora — mais uma venda fechada!

---

## Armadilhas Comuns

- ❌ **Criar oportunidade e nunca mais mexer**: pipeline fica desatualizado
- ❌ **Não registrar interações**: perde-se o histórico do relacionamento
- ❌ **Probabilidade irrealista**: 50 oportunidades em "Fechado (Ganho)" mas nenhuma fecha
- ❌ **Ignorar oportunidades perdidas**: sem análise de causa, erros se repetem
- ❌ **CRM desvinculado do ERP**: cliente aparece no sistema de vendas mas não no financeiro
- ✅ **CRM só funciona se for alimentado diariamente** — crie o hábito
