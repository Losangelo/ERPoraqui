# Movimentações Internas — Guia de Uso

## O que é?

Movimentações Internas são **transferências de produtos** entre filiais da mesma empresa ou ajustes manuais de estoque (entradas e saídas sem nota fiscal).

---

## Tipos de Movimentação

| Tipo | Descrição |
|------|-----------|
| **Transferência** | Produto sai de uma filial para outra |
| **Ajuste de Entrada** | Corrige estoque para mais (sobra de inventário) |
| **Ajuste de Saída** | Corrige estoque para menos (perda, quebra, roubo) |
| **Consumo Interno** | Produto usado internamente (ex: material de escritório) |

---

## Passo a Passo

### 1. Transferência entre Filiais

1. Acesse **Estoque > Movimentações**
2. Clique em **+ Nova Movimentação**
3. Selecione o tipo **Transferência**
4. Escolha:
   - **Origem**: filial de onde sai o produto
   - **Destino**: filial para onde vai o produto
5. Adicione os produtos com quantidades
6. Clique em **Efetivar**
7. O estoque é reduzido na origem e aumentado no destino

### 2. Ajuste Manual

1. Clique em **+ Nova Movimentação**
2. Selecione **Ajuste de Entrada** ou **Ajuste de Saída**
3. Selecione a **filial**
4. Informe o **motivo** (campo obrigatório)
5. Adicione os produtos
6. Efetive

### 3. Estornar Movimentação

Se uma movimentação foi feita errada:

1. Localize a movimentação
2. Clique em **Estornar**
3. Confirme — o sistema reverte as quantidades

---

## Dicas e Boas Práticas

- **Sempre informe o motivo** em ajustes — essencial para auditoria
- **Transferências** são a forma correta de mover estoque entre filiais
- **Consumo interno** deve ser usado para produtos de uso administrativo
- **Evite ajustes diretos** — prefira inventário para correções

---

## Problemas Comuns

| Problema | Solução |
|---------|---------|
| Transferência não efetiva | Verifique se há estoque na origem |
| Ajuste sem motivo | Edite e adicione a justificativa |
| Produto não aparece na origem | Produto pode não estar cadastrado na filial de origem |
