## Como Usar Este Manual

1. **Inicie pelos módulos que você usa no dia a dia**
2. Cada guia tem: conceitos, passo a passo, dicas, boas práticas e armadilhas
3. Use a seção **"Dicas e Truques"** para atalhos e recomendações
4. A seção **"Armadilhas Comuns"** mostra erros frequentes para evitar

---

## Boas Práticas Gerais

- **Explore o sistema** sem medo — dados de teste podem ser criados e removidos
- **Input Hints**: todo campo tem um placeholder descritivo — leia antes de preencher
- **Atalhos de teclado**: use `Ctrl+K` para abrir a paleta de comandos
- **Temas**: altere entre claro/escuro/sistema no menu do usuário
- **Ajuda**: o menu Ajuda no sistema tem informações atualizadas
- **Dúvidas?** Consulte o Manual Técnico (acesso restrito a devs) ou pergunte ao suporte
