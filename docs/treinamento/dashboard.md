# Dashboard (Painel Inicial) — Guia de Uso

## O que é?

O Dashboard é a **primeira tela** que você vê ao fazer login no ERPoraqui. Ele exibe indicadores, gráficos e atalhos para as principais funções do sistema, dando uma visão rápida do estado do negócio.

---

## Componentes do Dashboard

### Cards de Indicadores (KPI)

No topo, cards com números resumidos:

| Indicador | Descrição |
|-----------|-----------|
| **Vendas do Dia** | Total vendido hoje |
| **Contas a Receber** | Valor total em aberto |
| **Contas Vencidas** | Valor total atrasado |
| **Estoque Baixo** | Produtos abaixo do mínimo |
| **Saldo em Caixa** | Saldo atual do fluxo de caixa |

### Gráficos

- **Vendas por Período**: gráfico de barras/linha com vendas dos últimos 7/30 dias
- **Contas a Receber vs Pagar**: comparativo do mês atual
- **Produtos Mais Vendidos**: top 10 produtos do período

### Atalhos Rápidos

Links diretos para ações frequentes:
- + Novo Pedido de Venda
- + Nova Conta a Receber
- + Novo Produto
- Abrir PDV

### Alertas

Notificações visuais sobre:
- Contas vencendo hoje
- Certificado digital próximo do vencimento
- Licença próximo do vencimento
- Estoque crítico

---

## Personalização

O Dashboard pode ser **personalizado** por usuário:

1. Clique em **Personalizar** no canto superior direito
2. Selecione quais indicadores exibir
3. Reordene os blocos
4. Salve

---

## Dicas

- **Configure os atalhos** para as funções que você mais usa
- **Acompanhe os indicadores diariamente** para tomar decisões rápidas
- **Vendas do dia** atualiza em tempo real
- **Alertas vermelhos** exigem ação imediata

---

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Gráfico não carrega | Verifique se há dados no período |
| Indicador mostra zero | Pode ser falta de movimentação no período |
| Dashboard lento | Muitos dados — use filtros de período |
