# Promoções — Guia de Uso

## O que é?

O módulo de Promoções permite criar **ofertas temporárias** com preços especiais. Ideal para liquidações, queima de estoque, Black Friday, datas sazonais e estratégias de marketing.

---

## Tipos de Promoção

| Tipo | Descrição | Exemplo |
|------|-----------|---------|
| **Percentual** | Desconto percentual sobre o preço | 20% off em todos os calçados |
| **Valor Fixo** | Preço fixo promocional | Camiseta por R\$ 29,90 |
| **Leve X Pague Y** | Compre quantidade, ganhe desconto | Leve 3 pague 2 |

---

## Passo a Passo

### 1. Criar uma Promoção

1. Acesse **Estoque > Promoções**
2. Clique em **+ Nova Promoção**
3. Defina:
   - **Nome**: identificação interna (ex: "Black Friday 2026")
   - **Tipo**: Percentual, Valor Fixo, Leve Pague
   - **Data Início / Data Fim**: período da promoção
   - **Descrição**: texto para exibição no PDV/carrinho (opcional)
4. Selecione os **produtos** participantes
5. Configure os valores (percentual ou valor fixo por produto)
6. Ative a promoção

### 2. Regras de Funcionamento

- A promoção só é aplicada **dentro do período** definido
- Fora do período, o preço normal é restaurado automaticamente
- É possível ter **múltiplas promoções** simultâneas
- Em caso de conflito, a promoção com **maior desconto** prevalece

### 3. Aplicação no PDV e Vendas

- No **PDV**: o preço promocional aparece automaticamente no carrinho
- No **Pedido de Venda**: o vendedor vê o preço promocional ao adicionar o item
- O sistema registra qual promoção foi aplicada para relatórios

---

## Dicas e Boas Práticas

- **Planeje com antecedência**: crie a promoção dias antes de ativar
- **Teste antes**: ative em um ambiente de teste para verificar os preços
- **Acompanhe resultados**: use relatórios de vendas por promoção
- **Limite por cliente**: defina quantidade máxima por cliente se necessário
- **Promoções sazonais**: crie modelos para datas recorrentes (Natal, Dia das Mães)

---

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Preço promocional não aparece | Verifique se a data início já passou e a promoção está ativa |
| Desconto aplicado errado | Confira se o tipo (percentual/valor fixo) está correto |
| Promoção expirou antes do previsto | Verifique a data fim — ajuste se necessário |
