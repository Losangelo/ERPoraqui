# Convênios e Licitações

## Convênios

Acordos formais com clientes para prestação de serviços ou fornecimento.

### Situações

| Situação | Descrição |
|----------|-----------|
| ATIVO | Convênio vigente |
| SUSPENSO | Temporariamente interrompido |
| ENCERRADO | Finalizado |

### Como Criar

1. Acesse **Fiscal > Convênios** ou busque "Convênios" no Ctrl+K
2. Clique em **Novo Convênio**
3. Preencha:
   - **Cliente**: busque com LookupField
   - **Número**: identificador único
   - **Descrição**: objeto do convênio
   - **Data Início**: início da vigência
   - **Data Fim**: opcional
   - **Valor Total**: montante acordado
   - **Observações**: informações complementares
4. Salve

## Licitações

Processos de compras públicas e privadas.

### Tipos

| Tipo | Descrição |
|------|-----------|
| PREGAO | Modalidade para bens comuns |
| CONCORRENCIA | Para valores elevados |
| CONVITE | Para valores menores |
| TOMADA_PRECOS | Para serviços de engenharia |

### Situações

| Situação | Descrição |
|----------|-----------|
| EM_ANDAMENTO | Em fase de análise |
| GANHA | Licitação vencida |
| PERDIDA | Não vencedora |
| CANCELADA | Processo cancelado |

### Como Criar

1. Acesse **Compras > Licitações** ou busque "Licitações" no Ctrl+K
2. Clique em **Nova Licitação**
3. Preencha:
   - **Órgão**: entidade promotora
   - **Tipo**: PREGAO, CONCORRENCIA, CONVITE ou TOMADA_PRECOS
   - **Descrição**: objeto da licitação
   - **Data Abertura**: data da sessão
   - **Valor Estimado**: orçamento previsto
   - **Itens**: adicione produtos participantes
4. Salve

### Boas Práticas

- Registre todas as licitações, mesmo as perdidas
- Atualize a situação assim que houver resultado
- Use observações para anotar concorrentes e valores
