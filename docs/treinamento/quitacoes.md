# Quitações em Lote — Guia de Uso

## O que é?

O módulo de Quitações permite **baixar (quitar) múltiplas contas a receber ou a pagar de uma só vez** em lote. Ideal para recebimento de faturamento do dia, pagamento de múltiplos boletos, ou conciliação em massa.

---

## Quando Usar

- **Fechamento do caixa do PDV**: quitar todas as vendas do dia de uma vez
- **Pagamento de fornecedores**: pagar várias notas fiscais do mesmo fornecedor
- **Conciliação bancária**: baixar contas que constam como pagas no extrato
- **Cobrança recorrente**: receber mensalidades de dezenas de alunos/clientes

---

## Passo a Passo

### 1. Quitar Múltiplas Contas a Receber

1. Acesse **Financeiro > Quitações**
2. Selecione a aba **Contas a Receber**
3. Use os filtros para localizar as contas:
   - **Cliente**: filtra por cliente específico
   - **Período**: data de vencimento
   - **Status**: EM_ABERTO, ATRASADO
4. Marque as contas desejadas (checkbox)
5. Clique em **Quitar Selecionadas**
6. Escolha a forma de pagamento (pode ser diferente para cada conta)
7. Confirme — todas as contas são baixadas simultaneamente

### 2. Quitar Múltiplas Contas a Pagar

1. Acesse **Financeiro > Quitações**
2. Selecione a aba **Contas a Pagar**
3. Filtre as contas desejadas
4. Marque as contas
5. Clique em **Quitar Selecionadas**
6. Informe dados de pagamento
7. Confirme

### 3. Agendamento de Quitação em Lote

Para pagamentos recorrentes:

1. Clique em **Agendar Quitação**
2. Selecione as contas
3. Defina a data de pagamento
4. Sistema processa automaticamente na data agendada

---

## Dicas e Boas Práticas

- **Sempre confira o total** antes de confirmar a quitação em lote
- **Use filtros** para não quitar contas erradas
- **Exporte a lista** para Excel antes de quitar para conferência
- **Quitações são irreversíveis** — confirme apenas quando tiver certeza

---

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Conta já paga aparece na lista | Use o filtro de status para mostrar apenas "EM_ABERTO" |
| Valor total não fecha | Verifique juros, multas ou descontos não aplicados |
| Conta de outra filial aparece | Selecione a filial correta nos filtros |
